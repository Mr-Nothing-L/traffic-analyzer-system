Traffic Analyzer · 高速交通事件判别系统

基于多模态大模型 (VLM) 的高速公路监控视频 10 类事件检测框架 · YAML 配置驱动 · 三种检测模式

Pipeline · 5 阶段流水线

  • • preprocessing — 视频预处理 (~12s)
  • • scene_understanding — 场景理解 (~30s)
  • • event_detection — 事件检测 (~50s 并发)
  • • post_processing — 跨事件推断
  • • report_generation — 报告 + 二进制位
  • • 总耗时 ~190s · 逐步骤计时输出

Detection · 三种检测模式

  • • direct_vlm — 单次 VLM 直接判断 (4 事件)
  • • logic_chain — YAML 多步推理链 (3 事件)
  • • scene_tag — 零 VLM · 标签推断 (3 事件)
  • • 二进制编码 10 位 {b0_b1_..._b9}
  • • is_active=false 关闭事件不改位

Configuration · 配置驱动

  • • event_categories.yaml — 事件 + 模式
  • • logic_chains.yaml — 多步链定义
  • • prompt_templates.yaml — Jinja2 模板
  • • cross_event_inference_rules — 推断
  • • .env — Provider · API Key 配置
  • • 新增事件无需改代码
